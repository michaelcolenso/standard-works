# Transformation Map — PNW Foraging & Mycology Field Guide + Log System

## Before → After (buyer state change)

| Dimension | Before purchase | After purchase |
|---|---|---|
| **ID confidence** | "I think this is a chanterelle but I'm 70% sure — I'll post to Reddit when I get signal." | "I ran the 3-step ID gate in the guide and it passed all three checks. I'm confident." |
| **Spot management** | "I found a great patch last October — I think it was near the north trailhead, maybe mile 3? I can't find my note." | "My spot log shows GPS coordinates, elevation (1,400 ft), peak date (Oct 8), and 2.3 lbs harvested last year." |
| **Season awareness** | "I go out for chanterelles. I don't really know what else is out there." | "My seasonal calendar shows Hedgehogs and Matsutake are peaking this week at my elevation — I'll check those habitats too." |
| **Safety protocol** | "I skip anything I'm not 100% sure about." or "I eat it if it looks right and nothing bad has happened yet." | "I use the pre-harvest checklist before eating any new species. If I can't check all 3 gates, I don't eat it." |
| **Data organization** | Notes spread across phone camera roll, paper notebook, and a Google Sheet I made in 2022 that I can't find. | All finds in one pre-built spreadsheet with species, location, date, elevation, weight, and notes in structured columns. |
| **Recipe knowledge** | "I know chanterelles are good sautéed. I have no idea what to do with Matsutake." | "Quick-ref card shows top 3 prep methods and 1 preservation technique per species." |

## Transformation sequence (how the product works)

**Step 1 — Before the foray (15 min)**
Open Seasonal Calendar → identify which 3–5 species are in season at your planned elevation → check Harvest Log for any prior spots at that elevation → plan route.

**Step 2 — In the field (3–5 min per specimen)**
Find specimen → open PDF to species section (organized by habitat/season, not A–Z) → run 3-step ID gate → if all 3 pass: confirm edible; if any fail: photograph and leave → fill in Harvest Log entry (species, GPS, elevation, date, weight).

**Step 3 — After the foray (5 min)**
Complete Harvest Log entry with notes → check Quick-Ref card for any new species found → update Spot Tracker if new location identified.

**Step 4 — Season planning (once per year)**
Review prior year Harvest Log → identify top 5 spots by yield → cross-reference Seasonal Calendar → set outing dates for next season.

## Asset-to-transformation mapping (5-Asset Rule)

| Asset | Transformation it enables |
|---|---|
| 1. Harvest Log (Master Tracker) | Spot memory + data organization |
| 2. Pre-Harvest Safety Checklist (Go/No-Go Gate) | ID confidence + safety protocol |
| 3. Seasonal Calendar (Sequence Checklist) | Season awareness + trip planning order |
| 4. Recipe & Preservation Quick-Ref (Worksheet) | Post-harvest utility + recipe knowledge |
| 5. Species Profile Verification Sheet (Verification) | Adversarial self-check: buyer confirms they ran all ID steps correctly |
