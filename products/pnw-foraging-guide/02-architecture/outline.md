# Outline — PNW Foraging & Mycology Field Guide + Log System

## Deliverable 1: Core Field Guide PDF (target: 60 pages)

### Front matter (pages 1–6)
- Cover + product overview (1 page)
- How to use this guide — reading the ID gate, safety rules, offline use instructions (2 pages)
- Mandatory disclaimer: supplement to, not replacement for, field mentorship; Poison Control number prominent (1 page)
- PNW Habitat Map — coast, lowland forest, mountain/subalpine, urban edge (1 page)
- Master Seasonal Calendar — all 50 species by month + elevation band (1 page)

### Section 1: Spring Foraging — March through May (pages 7–20)
Species covered (estimated 10–12):
- Morel (*Morchella* spp.) — 3-step ID gate, False Morel look-alike comparison, elevation and host-tree guide
- Stinging Nettle (*Urtica dioica*) — ID gate, harvest timing, preparation note
- Fiddlehead Fern (*Matteuccia struthiopteris*) — ID gate, look-alike (toxic Bracken comparison)
- Miner's Lettuce, Chickweed, Wood Sorrel, Spring King Bolete (elevation-dependent)
- Oyster Mushroom (*Pleurotus ostreatus*) — cold-weather flush; substrate identification
- Plus 3–4 additional spring species by frequency ranking

Each species profile format (1 page each):
1. Common name + scientific name + toxicity rating (Edible / Edible with prep / Avoid)
2. **3-Step ID Gate** (binary yes/no questions; if any = NO → do not eat)
3. Look-alike table (1–2 most dangerous look-alikes, key distinguishing feature)
4. PNW habitat and season window
5. Sustainability rule (e.g., "Harvest no more than 1/3 of visible fruiting bodies per patch")
6. One-line prep note

### Section 2: Summer Foraging — June through August (pages 21–33)
Species covered (estimated 10–12):
- Chanterelle (*Cantharellus formosus*) — primary target; Jack O'Lantern comparison
- Chicken of the Woods (*Laetiporus conifericola*)
- Lobster Mushroom (*Hypomyces lactifluorum*)
- Elderberry (*Sambucus nigra*) — unripe toxicity warning; preparation requirement
- Blackberry, Salal berry, Thimbleberry
- Summer King Bolete (higher elevations)
- Plus 3–4 additional summer species

### Section 3: Fall Foraging — September through November (pages 34–50)
Species covered (estimated 15–18; peak season, highest species diversity):
- King Bolete / Porcini (*Boletus edulis*) — Bitter Bolete comparison
- Matsutake (*Tricholoma magnivelare*) — Smith's Amanita comparison; high value, high care
- Hedgehog Mushroom (*Hydnum repandum*) — beginner-friendly; no dangerous look-alikes
- Cauliflower Mushroom (*Sparassis radicata*)
- Lion's Mane (*Hericium erinaceus*)
- Puffballs (Giant and Gem-studded) — interior cross-section check required
- Black Trumpet (*Craterellus cornucopioides*)
- Pig's Ear (*Gomphus clavatus*)
- Yellowfoot (*Craterellus tubaeformis*)
- Cranberry, Rosehips, Oregon Grape (late-season plants)
- Plus 5–6 additional fall species

### Section 4: Winter & Year-Round (pages 51–56)
Species covered (5–6):
- Velvet Shank / Enoki (*Flammulina velutipes*)
- Turkey Tail (*Trametes versicolor*) — medicinal/tea use
- Chaga (*Inonotus obliquus*) — host tree requirement; sustainable harvest
- Cattail, Dandelion (year-round/winter plants)

### Back matter (pages 57–60)
- Poisonous species never-eat gallery — 8 PNW species with photos and brief ID notes (Amanita phalloides, Amanita bisporigera, Gyromitra esculenta, Galerina marginata, etc.)
- Resources and further reading (Trudell/Ammirati, PSMS, NAMA, Washington Poison Center)
- Quick Start instructions for Harvest Log and templates
- Glossary (15 terms: spore print, mycelium, substrate, annulus, volva, etc.)

## Deliverable 2: Harvest Log (Excel + Google Sheets)

Pre-built spreadsheet with the following sheets:

### Sheet 1: Find Log (primary daily use)
Columns: Date | Species | Common Name | Location Name | GPS Coordinates | Elevation (ft) | Habitat Type | Substrate | Harvest Weight (lbs) | Confidence Level (1–3) | Notes | Photo Reference

### Sheet 2: Spot Tracker
Columns: Spot Name | GPS Coordinates | Elevation | Habitat | Best Species | Best Month | Peak Years | Notes | Return Priority (High/Med/Low)

### Sheet 3: Species Running Tally
Auto-tallied from Find Log: Species | Total Harvest (lbs) | # Visits | First Find Date | Best Find Date | Best Location

### Sheet 4: Annual Planner
Month-by-month view: which species are in season, which spots to hit, column for planned outing dates
