# Go/No-Go Contract Gate (Checklist)

Purpose:
Decide whether you are ready to sign a construction contract.

Steps:
Answer Yes/No. Count No’s:
- 0–1 No → GO
- 2–3 No → WAIT
- 4+ No → PRECON

Example checklist (10):
1) Scope snapshot written (1–2 pages)
2) Bid-ready drawings/specs exist
3) Permit path documented in writing
4) Unknowns list exists with owners/dates
5) Schedule includes inspection dependencies
6) Milestones tied to verification evidence
7) Upfront payment ≤10% unless documented long-lead procurement
8) Change order process written
9) Allowances defined with deadlines
10) Lien releases included per payment

Verification:
You can point to evidence for each Yes.
