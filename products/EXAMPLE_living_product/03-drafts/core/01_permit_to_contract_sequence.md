# Permit-to-Contract Sequence (Core)

Outcome:
Choose the correct sequence and avoid signing contracts before the permitting path is clear.

Steps:
1) Document permit path and constraints.
2) Freeze scope to bid-ready level before competitive bidding.
3) Collect bids against identical scope.
4) Run the Go/No-Go Contract Gate before signing.
5) Use milestone payments tied to verification checkpoints.

Example:
If a zoning variance is uncertain, use a preconstruction agreement first, then convert to construction after approvals.

Failure modes:
- Signing fixed price on concept drawings → change orders explode.
- Bidding before permit constraints are known → redesign after bids.

Verification:
- Two bids are within ~10–15% and describe comparable scope.
- Permit path is documented in writing (email/screenshot + tracker entry).
