# Prompt

Use this file as the immutable project specification.

## Goals
- Create a shippable knowledge product from ideation to sale.
- Keep buyer scope narrow and outcome-focused.

## Non-goals
- Broad, generic content without proof of buyer spend.

## Hard constraints
- Work phase-by-phase with no skipping.
- Keep decisions and status externalized in markdown.
- Repair validation failures before advancing.

## Deliverables
- Complete phase folders from 00-brief to 07-launch.
- Asset pack and packaged PDFs.
- Launch plan and objection handling docs.

## Done when
- `scripts/status_check.py` passes.
- `scripts/quality_gate.py` passes.
- Launch assets exist and can be used to sell the product.
