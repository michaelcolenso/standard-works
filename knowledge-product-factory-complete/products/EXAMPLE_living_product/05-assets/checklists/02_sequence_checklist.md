# Sequence Checklist

Purpose:
Enforce correct order of operations.

Steps:
1) Scope snapshot
2) Permit path documentation
3) Bid-ready set
4) Comparable bids
5) Go/No-Go gate
6) Milestone schedule + verification plan
7) Start with evidence collection plan in place

Example:
If bids vary wildly, pause and tighten scope rather than picking cheapest.

Verification:
Each step produces an artifact in your tracker.
