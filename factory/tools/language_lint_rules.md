# Language Lint Rules

## Disallowed phrases (unless immediately followed by a decision rule or example)
- "should consider"
- "generally"
- "it depends" (without a decision tree)
- "might want to"
- "typically"
- "often"
- "maybe"

## Preferred pattern
- Decision: **If [condition], then [action].**
- Example: include numbers, concrete terms, and sample outputs.
- Verification: how the buyer knows they did it right.
